## 一、竞争格局：锂矿行业与铯铷赛道的双轨定位

### 1.1 锂矿行业：中矿资源是"小而美"的差异化玩家

中矿资源在锂矿行业中并非规模龙头，而是以"铯铷垄断+锂矿自给率100%+多金属平台"为特色的差异化竞争者。

## 二、商业模式与护城河

### 2.1 商业模式："三驾马车"结构

中矿资源已经从地勘服务公司转型为**多金属资源平台**，形成三大业务板块：

```
铯铷盐业务（护城河）── 毛利率85%+，年净利润约20亿元，穿越周期
    +
锂盐业务（增长期权）── 2025年营收占比48.87%，受益于锂价回升
    +
铜矿/锗镓业务（远期期权）── 赞比亚铜矿2026Q4试产，2028年满产3.8万吨/年
```

### 2.2 护城河五维分析

#### 维度一：源头资源垄断（天赋型护城河，不可复制）
- 全球铯榴石资源约80%在手，唯一天赋型独立大矿
- 锂矿自给率100%（Bikita + Tanco），不受外部原料供应制约
- 总锂资源量：约270-289万吨LCE（Bikita 1.41亿吨矿石/343.41万吨LCE + Tanco）

#### 维度二：全产业链闭环
- 铯铷：采矿--加工--精细化工--甲酸铯租赁（全球唯一）
- 锂：采矿--选矿--锂盐冶炼--氟化锂（进入特斯拉供应链）
- 产品壁垒：甲酸铯等产品在石油钻井、航空航天、军工领域形成绝对锁定

#### 维度三：成本位置
- 锂盐完全成本：6.5-8万元/吨（含津巴布韦资源税、出口税、集团管理费用分摊）
- 直接生产成本：5-6万元/吨
- 硫酸锂厂建成后目标：4.5-6.0万元/吨
- 成本位置：全球锂成本曲线中游偏上（非洲运输成本高是主要拖累）
- 降本关键变量：津巴布韦一期3万吨/年硫酸锂项目（2027年投产），二期再增3-4万吨

#### 维度四：技术壁垒
- 率先掌握透锂长石单烧/混烧生产电池级碳酸锂/氢氧化锂的工艺
- 电池级氟化锂生产工艺获发明专利，产品进入特斯拉供应链
- 国家级科研项目"含锂瓷土矿低品位共伴生铷铯超常富集"通过中期验收
- 已申请发明专利22件，研发出4N级铷铯盐5种，计划合成5N级超高纯产品

#### 维度五：全球客户绑定
- 铯铷盐精细化工产品长期供应全球头部客户（油气、半导体、军工）
- 甲酸铯钻井液：全球唯一"租赁+回收"模式，服务粘性极强
- 锂盐客户包含特斯拉等头部电池/车企

## 五、行业前景

### 5.1 锂行业：供需紧平衡，价格中枢上移

| 预测来源 | 关键判断 |
|----------|----------|
| 摩根士丹利 | 2026年全球锂市场出现约10万吨供需缺口，库存不足一个月需求量 |
| 瑞银 | 2026年锂需求增速14%，2027年升至16% |
| 五矿证券 | 2025年锂价已从底部反弹，行业景气度逐步修复 |
| 行业共识 | 2026年碳酸锂市场从结构性过剩转向"全年紧平衡、局部短缺" |

**需求驱动**：储能（增速最快）+ 动力电池（稳健增长，2026年全球电动车渗透率持续提升）

**2026年碳酸锂价格中枢**：约15-16万元/吨（含税），较2024年低谷（约7-8万）翻倍

### 5.2 铯铷行业：供需缺口即将爆发

#### 全球供需平衡表（东兴证券，2026年6月）

| 年份 | 全球供给（吨） | 全球需求（吨） | 供需平衡（吨） |
|------|---------------|---------------|----------------|
| 2026E | 3,183 | 3,166 | +16（小幅宽松） |
| 2027E | 3,870 | 4,599 | -729（首次缺口） |
| 2028E | 4,420 | 6,104 | -1,684（缺口扩大） |
| 2030E | 5,200 | 8,600+ | -3,400+（长期供不应求） |

**关键结论**：2026-2027年供需小幅宽松，价格平稳；**2028年起供需反转，进入长期供不应求周期**，产品价格具备翻倍弹性。

#### 需求爆发点

| 下游领域 | 增长驱动 | 增速 |
|----------|----------|------|
| **钙钛矿光伏** | 2026-2030年全球装机从20GW暴增至281.7GW | 铷盐需求CAGR 94% |
| **太空光伏** | 商业航天、卫星组网对钙钛矿电池需求 | 增量巨大 |
| **5G/6G基站** | 铷原子钟，2026-2030年全球5G宏基站铷盐消耗合计38.4吨 | 稳定增长 |
| **低轨卫星** | 中国低轨卫星铷原子钟需求2026年2,043台升至2030年28,989台 | 铷盐需求大增94% |
| **量子通信** | 2025-2030年全球铷盐需求量CAGR 33% | 高增速 |
| **传统领域** | 5G基站、半导体、航空航天 | 年增10-15% |

#### 钙钛矿的"铯铷催化剂"逻辑

- 每GW钙钛矿电池消耗铯铷盐约22吨
- 2024年钙钛矿太阳能电池市场渗透率仅0.5%
- 一旦产业化突破，铯铷需求将呈指数级增长
- 中矿资源是唯一的规模化铯铷供应商，几乎独占增量

### 5.3 铜矿：远期期权

- 赞比亚Kitumba铜矿：2026年Q4试生产，2028年满产约3.8万吨/年铜金属
- 铜价受全球能源转型+AI基建驱动，中长期趋势向上
- 铜矿是"第三增长曲线"，短期不贡献利润

### 5.4 锗镓：被忽视的小金属

- 纳米比亚锗镓项目：2025年Q4实现销售，从中间品升级为高纯终端品
- 锗镓是半导体、红外光学、光纤通信关键材料
- 中国对镓锗实施出口管制，海外资源价值凸显

## 六、风险因素

### 6.1 锂价波动风险（核心风险）

| 风险等级 | 高 |
|----------|-----|
| 描述 | 锂盐业务利润对锂价高度敏感。若碳酸锂价格大幅回调至10万元/吨以下，锂盐业务盈利能力将大幅压缩 |
| 当前状态 | 2026年Q1碳酸锂均价约15万元/吨，价格处于回升通道 |
| 2025年教训 | 全年锂盐价格低迷，导致归母净利润仅4.58亿元（-39.54%），充分暴露了锂价敏感性的风险 |

### 6.2 津巴布韦地缘政治与政策风险（正在发生）

| 风险等级 | 高 |
|----------|-----|
| 描述 | 2026年2月25日，津巴布韦矿业部宣布暂停所有原矿及锂精矿出口。津巴布韦2025年锂矿产量约占全球9.66%，是全球第四大锂矿生产国 |
| 对中矿资源的影响 | 2026年6-7月因津巴布韦锂精矿发货延误，导致江西锂盐厂临时停产检修（3万吨+3.5万吨两条产线） |
| 豁免情况 | 中矿资源完全满足津巴布韦政府出口禁令豁免条件，但执行层面仍有不确定性 |
| 长期影响 | 津巴布韦正推动产业链本土化，要求矿企在当地建设冶炼产能，中矿资源已启动硫酸锂项目应对 |

| 碳排放 | 全球碳边境调节机制可能增加出口成本 |
| 社会责任 | 海外矿山劳工权益、社区发展要求 |

### 6.8 产能扩张不及预期

- 津巴布韦一期硫酸锂项目（2027年投产）存在延期风险
- 二期扩产计划（3-4万吨）取决于一期进展和市场需求
- 赞比亚铜矿项目（2026Q4试产）在非洲基建环境中存在不确定性

## 九、核心结论

**中矿资源是全球铯铷资源的绝对王者，锂盐和铜矿业务以极低价格存在于当前股价中。**

**投资框架定位**：第二梯队上沿，适合作为战略卫星仓位。

**核心逻辑**："下有保底（铯铷垄断），上不封顶（锂铜期权+钙钛矿爆发）"。

**关键风险**：
1. 管理层资本配置瑕疵（估值折价，仓位上限约束）
2. 锂价波动（2025年教训深刻）
3. 津巴布韦政策风险（已引发停产）
4. 非洲铜矿投产延期
5. 钙钛矿无铯技术路线替代
