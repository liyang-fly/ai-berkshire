## 第二步：生意本质分析 — 段永平"对的生意"

### 一句话定义

> **铖昌科技做的是"太空基础设施的卖铲人"——卫星需要相控阵天线才能通信，相控阵天线需要T/R芯片才能工作，而铖昌是国内少数能做出宇航级T/R芯片的公司。**

### 收入结构拆解

```
星载（卫星）：~60%（2025年估计，增长最快）
弹载（导弹）：~25%
地面/其他：~15%
```

### 段永平式追问：这门生意好在哪？

**好在三个字：不可替代。**

1. **技术不可替代**：T/R芯片是相控阵雷达的"心脏"，一颗卫星需要数百到数千颗芯片，一旦定型验证（通常2-3年），几乎不可能更换供应商
2. **资质不可替代**：宇航级芯片需要特殊认证（抗辐射、高低温、真空等），国内通过认证的民营企业凤毛麟角
3. **时机不可替代**：中国卫星互联网刚起步，铖昌已占据先发位置

**但不够好的地方**：
- 客户高度集中，单一客户订单波动可导致2024年式的断崖（-26%）
- 产品非标准化，难以实现"可口可乐"式的持续复购
- 价格受制于军工审价体系，涨价空间有限

| 生意特征 | 评分（1-5） | 说明 |
|---------|------------|------|
| 客户锁定 | ⭐⭐⭐⭐ | 型号定型后极难替换 |
| 定价权 | ⭐⭐⭐ | 军工审价，但新产品有溢价 |
| 复购性 | ⭐⭐⭐ | 卫星量产带来持续需求，但周期波动大 |
| 可规模化 | ⭐⭐⭐⭐ | Fabless模式，轻资产扩张 |
| 护城河 | ⭐⭐⭐⭐ | 技术+资质双壁垒 |

## 第三步：护城河评估 — 巴菲特"经济护城河"

### 五类护城河逐一验证

| 护城河类型 | 评估 | 验证方法 |
|-----------|------|---------|
| **品牌/定价权** | ⭐⭐ | 军工采购不依赖品牌，依赖资质和关系 |
| **转换成本** | ⭐⭐⭐⭐⭐ | 宇航级芯片定型后，更换供应商需重新验证2-3年，成本极高 |
| **网络效应** | ⭐ | 不适用，芯片是物理产品 |
| **规模效应** | ⭐⭐⭐ | Fabless模式的规模效应有限，但研发费用摊销显著（收入越大，研发费率越低） |
| **技术/专利壁垒** | ⭐⭐⭐⭐ | GaN/GaAs工艺经验、宇航级可靠性设计Know-how，非短期可复制 |

### 护城河趋势

| 时期 | 护城河宽度 | 论据 |
|------|----------|------|
| 2018-2022 | 中等 | 技术领先但产品单一，客户集中 |
| 2023-2024 | 收窄 | 订单下滑暴露客户集中风险，毛利率从77%→62% |
| 2025-2026 | **快速拓宽** | 卫星互联网放量，星载产品占比提升至60%+，毛利率回升至82% |
| 2027-2030（预判） | **持续拓宽** | 只要中国低轨卫星计划持续推进，先发优势将转化为规模优势 |

### 巴菲特式追问：10年后这条护城河还在吗？

**在，但前提是中国卫星互联网计划持续推进。**

能摧毁护城河的因素：
1. 技术路线变革（如光子相控阵替代电子相控阵）
2. 中电科等军工所自研芯片替代外购
3. 卫星互联网计划被取消或大幅缩减

### 空方核心论点

1. **估值离谱**：TTM PE 260x，即使净利润翻倍仍有130x，远超全球军工芯片同行（通常30-50x）
2. **单一客户依赖**：前五大客户占比可能超80%，政治风险+订单波动
3. **2024年噩梦可能重演**：军工订单周期性强，2025年的爆发可能透支未来
4. **火箭回收与铖昌的关系间接**：发射成本下降→卫星制造需求增加，但传导链条长（3-5年），短期更多是情绪炒作

### 芒格式追问：我最可能在哪里犯错？

**最大的错误可能是：把周期当成长。**

2024年营收暴跌26%说明铖昌的生意本质上是**项目制+政策驱动**，而非"订阅制"的持续增长。2025年91%的增长中，有多少是订单回补，有多少是可持续趋势？如果2026-2027年再次出现订单真空期，300亿+的市值将面临剧烈回调。

**聪明人为什么不买？**
- 260x PE意味着即使每年利润翻倍，也要3年才能消化到合理估值
- 自由现金流为负，无法用DCF估值提供安全边际
- 军工业务的"黑箱"属性——无法验证订单真实性，只能相信财报

### 火箭回收的文明级意义（独家信息深化）

长征十号乙回收成功意味着：
1. **中国进入"可复用火箭"时代**，与SpaceX的差距从10年缩小到5-7年
2. **发射成本将指数级下降**：SpaceX Falcon 9复用后成本降至原来的1/3，中国有望实现类似降幅
3. **卫星数量将从"万颗级"进入"十万颗级"**：发射成本每降50%，经济可行的卫星数量翻倍
4. **铖昌的TAM将非线性增长**：不是存量替代，而是增量创造

### 李录式追问：站在20年后回看，铖昌科技是"标准石油"还是"3Com"？

**倾向"标准石油"的概率更高（约60%），但前提是：**

**成为"标准石油"的条件**：
- ✅ 中国低轨卫星计划持续推进（大概率）
